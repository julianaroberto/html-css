Principais formatações

Negrito/destaque

Nessa frase temos um termo em negrito usando a tag <b> (tag não semântica, não tem significado, apenas forma).

Nesata frasem tenho um termo em destaque usando a tag <strong> (tag semântica).

Itálico/Ênfase

Nesta frase, temos um termo em itálico com a tag <i> (não semântica).

teste para envelopar o texto com atalho ctrl+shift+p + tag.

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica.)

Texto marcado

Podemos criar também um texto marcado/em destaque usando a tag <mark>.

e no outro parágrafo, temos um outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e pequeno.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, com a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag <ins>.

Existe também a tag sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3 com a tag <sup>.

Texto subscrito

Para inserir coisas do tipo h20 com a tag <sub>.
